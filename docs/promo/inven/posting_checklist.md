# 인벤 게시 체크리스트

## 게시 톤

- 완성 서비스 홍보보다 “개인 개발 / 피드백 요청” 톤으로 작성합니다.
- 기능을 과하게 설명하지 말고, 닉네임 입력 → 최근 경기 분석 → 액션 제안 정도만 보여줍니다.

## 권장 게시판

- 1순위: FC온라인 인벤 팁과 노하우
- 2순위: FC온라인 인벤 자유게시판

## 게시 전 확인

- https://fcoach.fun 접속 가능 여부
- 닉네임 검색 1회
- 공식/친선, 5/10/30경기 선택 정상 동작
- 최신 경기 반영 시간이 너무 오래되지 않았는지 확인

## 댓글 대응

- 공식 서비스인가요?
  - 넥슨 Open API를 활용한 비공식 팬 서비스입니다.
- 승률이 바로 오르나요?
  - 보장하지 않습니다. 최근 경기 데이터를 보고 무엇을 먼저 고칠지 잡아주는 참고 도구입니다.
- 최신 경기가 안 보여요.
  - Open API 반영 지연 또는 호출 제한이 있을 수 있습니다.
