# 스크린샷 첨부 안내

첫 홍보글은 **3장만 올리는 것**을 권장합니다. 기능을 전부 보여주기보다 “가볍게 써볼 만한 사이트”로 보이는 편이 반응을 받기 좋습니다.

## 권장 업로드 순서

1. `simple/screenshots/01_start.png`
   - 닉네임 입력 후 빠른 시작
2. `simple/screenshots/02_dashboard.png`
   - 최근 경기 핵심 지표
3. `simple/screenshots/03_action_plan.png`
   - 다음 경기 액션 플랜

## 선택 이미지

- `simple/screenshots/04_player_report.png`
  - 선수 리포트까지 강조하고 싶을 때만 추가합니다.

## 업로드 팁

- 랭커 분석, 개선 추적, 이용 가이드는 첫 글에서는 빼는 편이 낫습니다.
- 스크린샷 설명은 이미지 위아래에 한 줄만 붙이면 충분합니다.
- “아직 개인 개발 MVP라 피드백 받고 싶다”는 톤을 유지하는 것이 안전합니다.
