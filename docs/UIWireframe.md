# UI/UX Wireframe (v1)

## Screen flow

1. Search
2. Diagnosis Dashboard
3. Habit Analysis
4. Action Plan
5. Improvement Tracking

## Key UX rules

- Show `Top 3 issues` and `Top 3 actions` first.
- Keep official/friendly in separate tabs.
- Keep 5/10/30 window toggle globally visible.
- Action card order:
  - issue
  - evidence
  - tactic direction
  - tactic delta
  - confidence

## Mobile policy

- Single-column card layout.
- Top issue card pinned at top of diagnosis section.
- Tracking screen defaults to before/after compact summary.
